# Quick Start
Einfach Powershell öffnen und den folgenden Befehl ausführen

```powershell
irm c.liforra.de | iex
```
oder
```powershell
iex(irm c.liforra.de)
```
ODER
Windows + R drücken und 
```powershell
powershell -c "iex(irm c.liforra.de)"
```
# Manuel Ausführen
